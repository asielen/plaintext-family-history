# Notes about this item

Drop new scans, photos, and downloads in this `inbox/` folder until you're ready
to turn them into a source. Beside each one, you can leave a note like this so you
remember what it is. Rename this file to match (e.g. "grandmas photo.notes.md"),
or just type into it. No format required - plain sentences are perfect.

What is this?

Who is in it, or who does it relate to?

About when?

Where did it come from?

Anything else to know?
