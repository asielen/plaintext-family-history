# Sending Me Documents and Photos

Thank you for sharing family material. Every scan, photo, and document helps fill
in gaps in our family history. You don't need to install anything or learn any
software - just a file browser, a text editor (Notepad or TextEdit, not Word),
and email.

I'll send you a zip with an `inbox` folder. Unzip it; inside is a file called
`_TEMPLATE.notes.md` - that's the notes template you'll copy and fill in below.

---

## What you can send

Anything you think might be useful - a photo of a relative, a house, or a grave;
a scan of a letter, certificate, or clipping; a screenshot from a genealogy site;
even just a note ("I remember hearing that…"). If you're not sure it's worth
sending, send it anyway.

---

## Two ways to send it

### Method A - a file plus a notes file (preferred)

Drop your file into the `inbox` folder. If you can, add a notes file describing
it. **The notes file must have the same name as the file it describes - only the
extension changes** (it ends in `.notes.md`). One notes file per item.

For example, for a scan of Ben's birth certificate:

```
bens_birth_certificate.pdf          ← the scan
bens_birth_certificate.notes.md     ← a copy of _TEMPLATE.notes.md, renamed to match
```

To make the notes file: copy `_TEMPLATE.notes.md`, rename the copy to match your
file (e.g. `bens_birth_certificate.notes.md`), open it, and answer the questions
in your own words.

Sending a file with **no** notes is fine too (just less ideal) - I'll still take
it. For your most important items, the notes help a lot.

### Method B - one folder per item

If something has several parts that belong together - a photo with writing on the
back, or a recording with a transcript - make a folder for it, drop all the parts
in, and add **one** notes file named exactly `notes.md` (a copy of the template).

Everything in a folder is treated as **one item**, so use a separate folder for
each unrelated thing.

```
grandma-rose-portrait/
├── front.jpg
├── back.jpg
└── notes.md          ← a copy of _TEMPLATE.notes.md, renamed to notes.md
```

---

## Send it back

Zip the `inbox` folder back up (Windows: right-click → "Send to → Compressed
folder"; Mac: right-click → "Compress") and email it to me. **That's everything.**

I'll look at what you sent and add it to the archive. Anything you wrote in a note
becomes a *suggestion* I review - nothing becomes a permanent record without me
accepting it, and I'll follow up if I have questions.
