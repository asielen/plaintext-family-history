Copy this file and fill it in. Rename the copy to match the file it describes
(e.g. `bens_birth_certificate.notes.md`), or to `notes.md` if you're putting one
item's parts in their own folder.
Write in plain sentences - no special format needed.
Delete these instructions if you like, or leave them; either is fine.

Don't worry about getting things "right":
- **Dates** can be approximate - "around 1955" or "probably the 1940s" is perfect.
- **Spelling** of names is fine as you know it; guesses help.
- **Don't rename your photo or scan** - use whatever name your device gave it.
- **You can't break anything.** Nothing you send overwrites or deletes what's
  already in the archive; everything is reviewed before it goes in.
