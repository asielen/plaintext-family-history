Type

Body and headings are set in Literata, a text serif. Fraunces is reserved for the masthead and person names. Dates use oldstyle figures in running prose and tabular figures in tables. Both faces are self-hosted; pages work offline.

Local overrides

Project-specific changes belong in custom.css, which every page links after styles.css. Change a few tokens to restyle every generated page, the tree, and exports at once.

:root {
  --paper:  #f4f1ea;               /* cooler paper            */
  --accent: #3e4a3a;               /* bottle-green links       */
  --font-serif: "Iowan Old Style", Georgia, serif;
}

/* Reach past tokens for finer control: */
h2 { border-bottom-width: 3px; }

The tokens are defined and commented at the top of styles.css: the paper/ink neutrals, the single accent, the earth branch set (--branch-1..7), the status inks, and the type (--font-serif, --font-display, the --text-* scale). Dark mode and an optional paper grain are marked in place, both off by default.
